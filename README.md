## Цель проекта (формулировка)

Проанализировать продажи интернет-магазина, выявить ключевые тренды во времени и сегменты клиентов на основе покупательского поведения.

Проанализировать покупательское поведение клиентов интернет-магазина, выявить ключевые сегменты клиентов и факторы, влияющие на выручку.


## Бизнес-вопросы (выбираем 3 — оптимально)

### Вопрос 1 — динамика
Как менялись продажи и выручка во времени?
(по месяцам / дням)

### Вопрос 2 — клиенты
Какие клиенты приносят наибольшую выручку?
(повторные vs разовые)

### Вопрос 3 — сегментация (ключевой)
Можно ли выделить сегменты клиентов по покупательскому поведению?
(база для RFM)


## План проекта

### Шаг 1. Загрузка и очистка данных

- пропуски
- отрицательные quantity
- отменённые заказы

### Шаг 2. Временная динамика

- выручка по месяцам
- количество заказов

### Шаг 3. Повторные покупки

- сколько клиентов покупали >1 раза
- средний чек повторных клиентов

### Шаг 4. Сегментация (RFM light)

- Recency
- Frequency
- Monetary

## Этапы проекта

### Этап 1. Очистка и подготовка данных

- возвраты
- аномальные цены / количества
- приведение CustomerID к int
- новые фичи: month, day, hour

### Этап 2. Анализ выручки

- общая выручка
- топ-страны
- топ-товары
- Pareto (20% товаров = % выручки)

### Этап 3. Анализ клиентов

- сколько клиентов
- средний чек
- частота покупок
- RFM-анализ ⭐ (очень ценится)

### Этап 4. Временной анализ

- динамика выручки
- сезонность
- пики продаж

### Этап 5. Выводы и рекомендации

- что приносит деньги
- кто лучшие клиенты
- куда бизнесу смотреть дальше
